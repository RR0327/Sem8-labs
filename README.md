# Sem8-labs

**Laboratory work and practical assignments from the eighth semester.**

This repository contains all lab experiments, implementations, and assignments completed during the eighth semester of study. It is organized by subject, with each subject folder containing individual lab sessions.

---

## Repository Structure

```

sem8-labs/
├── AI/ # Artificial Intelligence
│ ├── LAB_1/
│ ├── LAB_2/
│ └── ...
├── DIP/ # Digital Image Processing
│ ├── LAB_1/
│ ├── LAB_2/
│ └── ...
└── DM/ # Data Mining
├── LAB_1/
├── LAB_2/
└── ...

```

| Folder | Subject                  |
| ------ | ------------------------ |
| AI     | Artificial Intelligence  |
| DIP    | Digital Image Processing |
| DM     | Data Mining              |

Each subject folder contains numbered lab directories (LAB_1, LAB_2, etc.) holding the respective experiment files, code, and documentation for that session.

---

## What You Will Find Here

- Lab assignments with problem statements and solutions
- Source code and implementations
- Supporting documentation and reports
- Output samples and results where applicable

---

## Cloning This Repository

To get a local copy of this repository on your machine, run:

```bash
git clone https://github.com/RR0327/Sem8-labs.git
cd Sem8-labs
```

Once cloned, you can navigate into any subject folder and its corresponding lab directory to access the files.
