## Experiment Title:

**Missionaries and Cannibals Problem** using **Uninformed Search Algorithms.**

## Problem Statement

Implement the **Missionaries and Cannibals problem** using **two uninformed search algorithms**:

- Breadth-First Search (BFS)
- Depth-First Search (DFS)

### Objective

To solve the **Missionaries and Cannibals problem** using Breadth-First Search (**BFS**) and Depth-First Search (**DFS**), and compare their performance in finding a valid solution while ensuring all problem constraints are satisfied.
