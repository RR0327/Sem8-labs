## Experiment Title

Implementation of the **8-Puzzle Problem** Using **Uninformed Search Algorithms.**

## Problem Statement

Implement the 8-Puzzle problem using **four uninformed search algorithms**:

- **Breadth-First Search (BFS)**
- **Depth-First Search (DFS)**
- **Depth-Limited Search (DLS)**
- **Iterative Deepening Search (IDS)**

> ### The **`objective`** is to transform the initial state into the goal state by moving the blank tile while generating valid successor states and displaying the solution path.
